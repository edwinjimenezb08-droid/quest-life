# QuestLife

QuestLife es una app mobile-first estilo RPG para gamificar disciplina y hábitos.

## Flujo actual

1. Pantalla inicial tipo login/selección de personaje (nombre + clase).
2. Inicio con información del avatar y estado del jugador.
3. Barra inferior con: Inicio, Misiones, Mazmorras, Tienda y Perfil.

## Sistema de misiones

- **Misión diaria fija**: siempre la misma todos los días; al completarla sube la racha.
- **Misiones del día (rotativas)**: cambian automáticamente según el día.
- **Misiones secundarias**: complementarias para progresar en atributos.

## Ejecutar localmente

```bash
python -m http.server 4173
```

Abrir en `http://localhost:4173`.
